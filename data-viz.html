Project Overview

This project visualizes global life expectancy trends using interactive charts built with D3.js, React, and Apache Echarts. The web application highlights changes in life expectancy across regions and time periods, offering insights into global health patterns. The application is deployed via GitHub Pages for public access.

Key Features

  • Interactive charts showcasing life expectancy trends from 1950 to 2021.
  • Comparisons by region, gender, and time period.
  • Dynamic bar charts, line charts, and world maps for easy exploration of global patterns.

Data Source

The data is sourced from the United Nations Population Division, covering life expectancy statistics across continents and countries from 1950 to 2021, broken down by gender.

Preprocessing

The original dataset (~24 MB) was filtered and cleaned to optimize performance for the visualizations. Unnecessary columns were removed, and data was grouped and sorted to focus on the key insights for each visualization.

Technical Details

  • Language: JavaScript
  • UI Framework: React
  • Charting Library: Apache Echarts
  • Deployment: GitHub Pages
