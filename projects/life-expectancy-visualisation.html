Life Expectancy Visualisation

React
Apache Echarts
JavaScript
Data Visualization

This project visualizes a comprehensive and interactive view of global life expectancy trends using data from the United Nations Population Division. The interactive web application aims to highlight changes in life expectancy across different regions and time periods, providing valuable insights for users.

Project Overview

The Life Expectancy Visualization project presents global health data in an interactive, web-based format. The application consists of five main slides, each offering different perspectives on life expectancy data:

Slide 1: Global Overview

Presents an overview of the average life expectancy worldwide, offering a baseline understanding with an engaging illustration that introduces the core concept of the website. Based on UN data for 2021, on average people are expected to live 71 years.

Slide 2: Historical Evolution

Showcases the evolution of life expectancy over the years, highlighting the top 10 countries with the highest life expectancy in each decade. This required special scripting to filter and group data by year and country.

Slide 3: Gender and Age Comparison

Compares life expectancy between genders and age groups, providing insights into demographic disparities.

Slide 4: Growth Analysis

Illustrates the growth in life expectancy from 1950 to 2021 as a result of medical advancements, with gender-based comparisons.

Slide 5: Geographic Distribution

Uses a world map with color coding to compare life expectancy across countries, facilitating clear visual comparison.

Dataset

The data comes from the United Nations Population Division and includes information on continents, countries, and life expectancy percentages for males, females, and both genders from 1950 to 2021. While the initial dataset was approximately 24 MB, it was optimized by removing unnecessary columns to create a more efficient version for the project.

Visualization Choices

Bar charts were chosen because the data is linear and not hierarchically related. They are ideal for comparing discrete categories or groups, making it easy to visualize differences and trends. Line charts were also used to provide clearer comparisons over time, as they effectively illustrate trends and changes in continuous data.

Technical Implementation

  • Language: JavaScript
  • UI Framework/Library: React
  • Deployment: GitHub Pages
  • Charting Library: Apache Echarts

Development Process

The project involved several key steps:

  1. Data preprocessing to clean and format the UN Population Division dataset
  2. Development of custom scripts to filter and group data for specific visualizations
  3. Implementation of interactive visualizations using Apache Echarts
  4. Integration of visualizations into a React application
  5. Deployment of the final application to GitHub Pages

Impact and Learning Outcomes

This project demonstrates the power of data visualization in making complex global health data accessible and understandable. It showcases skills in data processing, interactive visualization design, and front-end development.
